# Angular Material Framework
This Framework implements and Angular Module with helper components and services to provide a quick start in AngularJS single page applications using Material Design.

## Features 

#### Components
- amFramework
- sideMenu
- breadcrumbs
- toolbarButtons

#### Services
- loginDialog
- httpProgress

#### Animation
- slide
